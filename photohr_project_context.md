# PhotoHR — Полный контекст проекта
# Версия: 1.0 от 25.03.2026

## О проекте
HR-система для группы компаний LikeMe Studio.
Управление фотографами-продавцами и аквагримёрами на развлекательных площадках Москвы и СПБ:
океанариумы, концертные залы, театры, парки, торговые центры.

## Бизнес-контекст

### Схемы оплаты
- Standard: фикс за единицу + дневной бонус + месячная премия (свободный график)
- Premium: % от продаж, постоянный график 20 смен/мес, 2 выходных

### Бумажный отчёт площадки
Ежедневно заполняется на площадке:
- Счётчики принтеров (формат, начало/конец → напечатано)
- Таблица: фотограф × товар → количество
- Брак, флешки (шт + сумма), размен, ручка
- Разбивка оплат: нал/карта/QR
- Администратор + менеджер + подпись

### Жизненный цикл отчёта
draft → submitted → approved (→ payroll + рейтинги + списание)
                  → returned (→ доработка → submitted)
                  → rejected

### Рейтинг продаж
z-score внутри смены (сравнение с коллегами) → EMA сглаживание → rating_sales
+ rating_discipline (опоздания) → rating_overall

### Спецбонусы (5 типов автоусловий)
availability_on_time, total_sales_milestone, shifts_streak, no_late_month, tenure_months
